Item Properties

Label

A label to be used uniquely at the creator's disposal that does not affect the visual properties of the item.

Collapsed Bullet

Specifies the bullet image to be used when the item in question is collapsed and is not showing any children items it may have.

Expanded Bullet

Spcifies the bullet image to be used when the item in question is expanded and is showing any children items it may have.

Move Over Picture

A picture to be used that will display when the mouse is over the item in question.

Move Out Picture

A picture to be used that will display when the mouse is not over the item in question.

Text

Text to be displayed as the item, editing this will automatically change the label, edit in reverse to keep them different.

Font/Default

When Default is selected, the font specified in the main tree properties is use; else, selecting font will allow individual setting of the font. Unselecting both does not put any font info in for the item.

Link URL

Specifies the URL that this item should link to when clicked. This can be a full qualified http path or a single file relative to the folder the tree will be exported to.

Link Tool tip

Specifies the tool tip text to be displayed when you mouse over the specified item.

Link Target/Default

Specifies a link target to push the clicked content to, selecting default uses the tree's properties link target.
