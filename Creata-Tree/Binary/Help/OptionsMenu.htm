Options Menu

Graphical Editing

Selecting this causes the editor to use a more graphical view for editing, not every effect will show in response to this view.

Preview Pane

Selecting this causes the editor to have two separate views of the tree, the editing pane, and the actual HTML pane that shows what the tree will look like exported.

Force Dimensions

Selecting this causes the editor to force each tree item to a defined height and width that is set by 'Height of each Item' in the tree properties.

Use Templates

Selecting this causes a prompt to select from defined templates when you create a new tree.

Use Blank Tree

Selecting this causes any new tree to begin from a blank tree.

Media Library...

Opens the media library editor window.
