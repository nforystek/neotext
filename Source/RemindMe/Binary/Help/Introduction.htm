Introduction

The Main Window

RemindMe operations run as a service based application which means that there is a background application that runs which is administrated through the main window pictured above.

Service Control

Through the first toolbar button, (or through your start menu RemindMe programs folder), is where you can start or stop the RemindMe service. This must be running for operations to occur. When it's running; a stop icon over the gear will be visible, when it's not running, a play icon over the gear will be visible.
