Edit Menu

Undo

Undo the previous script edit changes.

Redo

Redo any undid script edit changes.

Cut

Cut the selected text.

Copy

Copy the selected text.

Paste

Paste the clipboard text contents at the cursor.

Select All

Select all the script text.

Delete

Delete the selected script text.

Comment

Comment the selected script text.

Uncomment

Uncomment the selected script text.

Fin or Replace...

Opens a dialog which allows the Find or Replace action on the script.
