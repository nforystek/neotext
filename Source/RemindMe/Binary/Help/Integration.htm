Scripting View

Script Setup

There is the option for using either VBScript or JavaScript to edit or add actions to be scripted by operations. Simply click "View Script" and select the script language you wish to use. Optionally RemindMe comments can be used to integrate execution entry using the wizard.

Script Scope

The editor saves a single page of code stored in the database which the service operates with. Upon update, the service is updated unless there is design time errors, runtime errors will report back just the same. Any line that is an error will highlight with red.

Script Debugging

Additionally minimal debugging can be achieved using the single like evaluation and execution entry. Simply click on the entry bar, and use type a "!" for a statement or type a "?" for evaluation, then enter your code and hit enter. Evaluation will return value.
