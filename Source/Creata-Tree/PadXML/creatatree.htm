Description:
Creata-Tree is a program that constructs JavaScript menus an trees available for use with your web site. The program has a nice interface and view's for you to layout items in a basic edit layout. Requires basic HTML skills to include the export in your website. The dynamic tree works in all major JavaScript enabled browsers and also employee's a nice static HTML layout of the tree for others. JavaScript gives developers the ability to dynamically alter the trees properties from any page on their site. Creata-Tree can even do simple menus with out a tree design.
