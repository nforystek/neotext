Framing Trees

This is a short example of placing your tree into a framed HTML view:

First off, make sure all your items have a Link Target set, in this example we will use "content" you can choose this.

Next, export your tree to a folder, use windows explorer to navigate to the folder. Create a new text document and open it.

Enter the following code, replacing "intro.htm" with your first or default document (most likely the first tree item): 

		<HTML>
		<HEAD><TITLE>Creata-Tree Example</TITLE></HEAD>

		<FRAMESET COLS="200,*">

			<FRAME NAME="menu" SRC="tree.html" SCROLLING=AUTO NORESIZE>
			<FRAME NAME="content" SRC="intro.htm">

		</FRAMESET>
		<NOFRAMES>
		Your browser doesn't support frames.
		</NOFRAMES>
		</HEAD>

Close the text document and rename it to Index.htm, then open it!
