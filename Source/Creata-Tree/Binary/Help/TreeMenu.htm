Tree Menu

New...

Starts a new tree, prompting to close or save any already open tree. Depending on your options, a black tree will be created or a prompt will show to select a template.

Open...

Opens an existing tree, prompting to close or save any already open tree. Tree files are of .tree extension. Only Creata-Tree tree files are supported.

Close

Closes any open tree returning the main window to its default state, prompting to close or save any already open tree.

Save

Saves the currently open tree to its file, if no file has been chosen yet, a prompt will ask for the file.

Save As...

Opens a prompt asking for a file to save the tree to.

Properties...

Opens the main trees properties window of the currently open tree.

Export...

Opens the export window and prompts for the trees export info.

(Recent Files)

Displays up to four recent tree files for quick opening, prompting to close or save any already open tree. If there have not been any recent files "(No Recent Files)" is displayed.

Exit

Exits Creata-Tree, prompting to close or save any already open tree.
