Tree Properties

Save Images in Tree File

Selecting this will cause Creata-Tree to include all media files in the saved .tree file. Upon loading a tree, any missing files used get imported into the Media Library. This has no effect for exported trees files.

Auto Size Bullet Images

Selecting this forces the bullet icons in use to be stretched to fit the required dimensions which would be square to the height. Otherwise the respective height and width of the image file for the bullet will used.

Top Left X Pixel Position, Top Left Y Pixel Position

These two properties defines a static X and Y coordinate to use of the Top most Left Most corner of the tree.

Height of Each Item

Defines a height restriction for each and every item, this is only enforced if 'Force Dimensions' is selected in the Options Menu.

Show Tree lines

Defines whether or not the Tree lines will be used in the tree and allows setting of the color to view.

Show Plus/Minus

Defines whether or not the Plus/Minus will be used in the tree and allows setting of the color to view.

Background Color

Defines a color to be used for the background of the HTML file the tree appears on.

Background Image

Defines an image to be used for the background of the HTML file the tree appears on.

Default Link Target

When specified all new items created under the tree will by default be created with this text property as its link target.

Default Font

When specified all new items created under the tree will by default be created with these font properties as its default font.
