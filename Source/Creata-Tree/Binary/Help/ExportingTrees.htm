Exporting Trees

Folder

Select the folder you wish to export your tree to, clicking the folder icon allows you to browse. This folder will be saved in your tree file for future reference. Note: A folder with the same name as your tree file will be created under the selected folder which will contain all your exported tree files.

Include customization in html file

Specifies whether the customization file should be included in the tree.html file. Otherwise there will be a custom.js created along with the tree.html. This option may be used to keep separate the custom in cases where you might wish to modify it on the fly.

Include tree code in html file

Specifies whether the two code files that make the tree function should be included in the tree.html file. Otherwise there will be a body.js and a head.js created along with the tree.html.

Export

Any HTML or JavaScript files related to the tree will be overwritten and the new HTML and JavaScript files will be created. The main file created will be tree.html, plus depending on your options, custom.js, body.js, and head.js.
