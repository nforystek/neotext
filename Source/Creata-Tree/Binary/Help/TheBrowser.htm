The Browser

Category

This window displays all the media Creata-Tree keeps track of; there are three categories, Bullet Icons, Menu Pictures, and Backgrounds.

Import

To use media in your tree you must first import it using the Import button. Supported image types are (BMP, GIF, JPG and PNG)

View

Views the selected image in a separate resolution window.

Delete

Removes the selected image from the image library. Note: Any tree using this item will not property view unless the media has been saved in the tree file.
