Item Menu

Add

Adds a new item to the tree under the selected tree or item.

Cut

Cuts the selected tree item placing it in the clipboard to be pasted elsewhere.

Copy

Copies the selected tree item placing it in the clipboard to be pasted elsewhere.

Paste

Pastes any tree item in the clipboard under the selected tree or item.

Move Up

Moves the selected item up the tree in organization.

Move Down

Moves the selected item down the tree in organization.

Remove

Removes the selected item from the tree.

Properties...

Opens the properties window of the selected tree item.
