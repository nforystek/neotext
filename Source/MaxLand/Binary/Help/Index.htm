MaxLand

Maxel is a game of environment challenges. In a desolate unknown land you are the queen of a palace carved into the side of a mountain by chess pieces and must discover Earth, Win, Fire and Water.

3D Controls

As written on the top of the screen during play is the controls which are as follows:
E=Forward, D=Back, W/R=Sideways, SPACE=Jump, TILDA=Console, ESC=Exit

E = Makes your player go forward.
D = Makes your player go backwards.
W/R = Makes your player move sideways.
SPACE = Makes your player jump up.
F1 = Toggles similar on screen help.
F2 = Toggle on screen debug info.
F3 = Resets the game, immediately.
 F5 = Cycle different view perspectives.
TILDA = Toggles visibility of the console.
ESC = Exits the focus and closes the game.

Console

To open the MaxLand console, run the game, and simply hit the tilda (~) key, hit it again to close the console. The console is used to operate and change options of the game.

HELP

Use HELP command gives information on using the console, the following text will display:

For detail help please type one of the following help commands:
      HELP COMMANDS (Displays the help of basic console commands)
      HELP EDITING (Displays the help of editing files in console)

HELP COMMANDS

Use HELP COMMANDS to view further informational help on the commands recognized by the console, the following text will display:

Console Commands:
      EXIT (completely exit out of the game)
      ECHO <Text> (Displays Text in the console)
      EVAL <Text> (Evalutes Text as a console command)
      FADE <Text> (Displays Text in the center of screen)
      DRAW <X> <Y> <Text> (Draw Text on the screen at X,Y)
      PRINT <Row> <Col> <Test> (Like DRAW but use row,col)
      CLEAR (Clears all text being drawn by DRAW or PRINT)
      STAT (Displays XYZ, Distance, Camera Angle and Pitch)
      LEVEL [<Title>] (Load a Title PX in the Levels Folder)
      RESET (Resets the game reloading everything like new)
      SPECTATE (Change in game involvement to a spectator)
      JOIN (Rejoins the game when yopur in spectator mode)

HELP EDITING

Use HELP EDITING to view commands that allow live editing of game data files during play, the following text will display:

Editing Commands:
      REFRESH (This command will cause the engine to reload the game files)
      LOAD <file> (This command loads the specified file to be edited live)
      VIEW <#[-#]> (This command displays the specified line numbers text)
      LINES <#[-#]> (This command adds lines # to the current loaded file)
      EDIT <#> <text> (This command changes the specified line numbers text)
      SAVE (This command saves any edited changes made with a loaded file)
      (For information on object and identifier editing see documentation)

EXIT

Exits the game, the ESC key will also exit the game (in windowed mode, hitting the ESC key twice exists the game, once frees the most from the game, if the mouse is freed and the console is open, the ESC key a second time will not close the program, while the movement controls are paused when the console is open, you may still type to the console).

ECHO <Text>

This command simply prints the supplied text to the console as the most recent message. There is no evaluation of variables with the use of this command, to display variable values evaluated, use this command in conjunction with the eval command. Example "eval echo $variable.value"

EVAL <Text>

This command will execute the supplied text and executes it as a command also evaluating any variable to its value before execution and not during.

FADE <Text>

This command draws the supplied text on the gaming screen at the location X and Y based on pixels. This command can be called multiple times and each time the text will remain on the screen until the clear command is used, reversing all calls to print and draw.

DRAW <X> <Y> <Text>

This command draws the supplied text on the gaming screen at the location Row and Col based on rows and columns defined by the size of the screen font. This command can be called multiple times and each time the text will remain on the screen until the clear command is used, reversing all calls to print and draw.

PRINT <Row> <Col> <Test>

This command reverses all calls to print and draw clearing any text that is currently being drawn on the gaming screen.

CLEAR

This command reveals your location (X, Y and Z coordinates), your distance to 0,0,0 (the start), the camera's angle, and the camera's pitch (when in third person mode).

STAT

This command will load the PX file of the Title supplied which must be under the Levels directory which is located in the program files directory of the game installation.

LEVEL [<Title>]

This command resets the current level to the initial stat as if it were loaded for the first time, resetting any standing personal saved information.

RESET

This command places you into a spectator mode of gaming which allows you to free-fly around but not interact with the level and game. This mode is also used to debug and place cameras for the camera view perspective mode.

SPECTATE

This command places you back into the game as a player from spectate mode.

JOIN

REFRESH

This command will refresh all the current game data from the game folder.

LOAD <file>

The load command takes one parameter or none, specifying the filename of a file located under the game data folder to be loaded. Once loaded the file can be viewed, edited and saved, if you type the command with out a parameter it shows the currently loaded file.

VIEW <#[-#]>

This command takes one parameter, either a single line number to be displayed, or a line number range between two numbers specified seperate by a single dash. The line numbers text of the current loaded file will print to the console.

LINES <#[-#]>

This command adds blank lines to the currently loaded file, it takes one parameter, either a single line number, or a line number range between two numbers specified seperate by a single dash. Blank lines will be added at the line location with out overwriting the line.

EDIT <#> <text>

This command takes two parameters, a single line number, and any text after it. The line number specified will be set to the text provided overwriting the line in the currently loaded file.

SAVE

This command takes no parameters, it simply saves the currently loaded file which should exist as the loaded file. To view the loaded filename you can use the load command with out any parameters.
