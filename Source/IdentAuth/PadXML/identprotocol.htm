Description:
IdentProtocol is a windows service that listens for incomming ident requests and responds with the proper info. It has a INI file for setting up the options and has no interface. The service itself works inconjunction with itself when more then one is running with in the same intranet and any requests to even misdirected will be correctly relayed to the outside inquery. This service responds with the Windows User account and optionally the computer name to the requester. It can also be set to act indenpendant or even not at all unless a spoecified program you supply is running.
