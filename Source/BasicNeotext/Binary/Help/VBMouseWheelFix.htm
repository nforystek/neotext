There is two parts of BasicNeotext that is running when you open a new instance of Visual Basic 6.0 one is a front loader as well as an Add-In which is all housed in the VBN.EXE application.  The second part of BasicNeotext is just an Add-In feature that is housed in BasicNTAddIn.dll.  Both are written with Visual Basic 6.0 Extensibility and can be found in the Visual Basic installation folder after installation of BasicNeotext.

 

An absent feature of the ability of Visual Basic IntelliSense resulted in Microsoft making a hot fix downloadable from their website which supported the use of the mouse wheel in code panes.  The hot fix file name is VB6MouseWheel.EXE (Singer information; Name: Microsoft Corporation, Signing time: Wednesday, August 31, 2005 3:23:23 PM)

 

Similarly, BasicNeotext houses a mouse wheel scroll fix in the BasicNTAddIn.dll that also stays aware of the initial hot fix so that it does not disturb the functionality should you already have it capable.
