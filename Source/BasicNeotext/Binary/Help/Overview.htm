Basic Neotext

 

Basic Neotext is an Add-In for Microsoft’s Visual Basic 6.0

 

1.      Features include code signing on compile seamlessly as Visual Basic use, command line or environment; strict ruling to what may as well be signed (requires Microsoft’s Authentication signtool.exe available in Windows SDK’s).

 

2.      Template for service applications providing needed debug restrictions and thorough propertied service information inclusion with no extra differences for Windows DOS based systems and Windows NT based systems.  Natural acting as well when with out forms just as any project would in unloading when no object remains.

 

3.      Debug and release advent of conditional compile variables for executables and inclusion of module variables expressed in conditional compiles so they may be used in declare aware inclusion/exclusion amongst multiple projects and shared declarations independently.

 

4.      A mouse wheel fix for Visual Basic code panes of which some versions and service packs did not have to able use of.
