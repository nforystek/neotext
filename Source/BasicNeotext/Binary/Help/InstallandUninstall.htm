BasicNeotext is a Visual Basic developed Visual Basic Add-in and enhancement suite.  The software requires you to already have, and installed, Microsoft’s Visual Basic 6.0.  BasicNeotext uses an installation system known as NSIS which also includes uninstall that can be accessible from the “Add/Remove Programs” in the control panel, or by the “uninstall.exe” file it will write upon completing the install.  When you run the download program file executable, you will be prompted to go through the installation wizard much like many of your usual programs may.  The first portion of the installer is two agreements, one for the warning and another for the open source agreement.  After agreeing to both consecutively, you will then be prompted to enter your Visual Basic 6.0 installation location, this is required and must be the folder to which the vb6.exe is located.  Usually this folder by default for Visual Studio 6.0 users is “C:\Program Files\Microsoft Visual Studio\VB98”.  The current build date of the installation is noted at the bottom of the install window during installation.
