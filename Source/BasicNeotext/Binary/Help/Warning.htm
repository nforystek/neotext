RISKS OF USE OF OPERATIONS’ WARNING

Alteration of project files, and source files of projects, will arise by the use of this Visual Basic 6.0 “add-in,” please use it at your own risk assessment preferably by exampled.  As well, it is a good habit and idea to have backups of your projects when examining adaptation software, as encouraged, before you commit to permanent use of automation which alters source applied to prior existing projects.  Installation requires Microsoft Visual Basic 6.0 to already be installed on this system, of which, must be selected as the installation folder during this wizard.  Small changes will be applied to the Visual Basic 6.0 installation with an able executable uninstall wizard located in the same path, plus accessible from the control panel’s “Add/Remove Program” applet.  Upon removal, alterations that remain in sources, do not affect normal use with VB6.
