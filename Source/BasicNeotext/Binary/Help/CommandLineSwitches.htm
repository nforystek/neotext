To see a visual lexicon of the logical command line switches (which are much the very same as VB6.EXE as they are in VBN.EXE) you can simply find the “About BasicNeotext Switches” menu item under the Visual Basic 6.0 “Help” menu, you may also start VBN.EXE located in the Visual Basic 6.0 installation folder with the switch “/?”   Either of these two options brings up the same dialog information acting just as VB6.EXE would, visually as well, pulling up a information window describing all the VB6.EXE feature command line switches as well as the differences, or additions, that the VBN.EXE has to offer.
