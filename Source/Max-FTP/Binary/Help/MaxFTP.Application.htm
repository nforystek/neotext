MaxFTP.Application ActiveX EXE



Identification

MaxFTP.Application

Description

This object is a MultiUse automation interface to the MaxFTP.EXE

Properties

AppProcessId (Long)

Description

The process ID of the Max-FTP application from the running executable.

AppThreadId (Long)

Description

The thread ID of the Max-FTP application from the running executable.

AppHInstance (Long)

Description

The handle instance of the Max-FTP application from the running executable.

AppComments (String)

Description

The comment of the Max-FTP application as is in the file properties.

AppCompanyName (String)

Description

The company name of the Max-FTP application as is in the file properties.

AppFileDescription (String)

Description

The file description of the Max-FTP application as is in the file properties.

AppLegalCopyright (String)

Description

The legal copyright of the Max-FTP application as is in the file properties.

AppLegalTrademarks (String)

Description

The legal trademark of the Max-FTP application as is in the file properties.

AppMajor (Integer)

Description

The version major of the Max-FTP application as is in the file properties.

AppMinor (Integer)

Description

The version minor of the Max-FTP application as is in the file properties.

AppProductName (String)

Description

The product name of the Max-FTP application as is in the file properties.

AppRevision (Integer)

Description

The version revision of the Max-FTP application as is in the file properties.

AppTitle (String)

Description

The title of the Max-FTP application as is in the file properties.

Functions

WindowCreate () (Long)

Description

Creates a new Max-FTP client window. Returns the handle to window.

WindowUnload (hWnd)

Parameters

hWnd Long

The windows handle to unload.

Description

Unloads the window handle provided that the handle is a reference to a window of Max-FTP.

WindowExists (hWnd) (Boolean)

Parameters

hWnd Long

The window handle to check if exists.

Description

Check the existence of a Max-FTP window by the window handle. Returns true if the handle is found as a Max-FTP window, otherwise false.

QuitApplication ()

Description

This closes out the Max-FTP application and any open windows.
