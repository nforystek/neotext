NTCypher10.LUID ActiveX DLL



Identification

NTCypher10.LUID

Description

This object is used to generate LUID's (local unique identifiers).

Enumerators

TimedSums

NoTiming = 0
CheckSums = 1
BoundSums = 2

Properties

Pacerate (TimedSums)

Description

Determines how the Generate function handles timing to creation of a LUID. NoTiming lets them occur at the nature of the systems top speed. CheckSums controls timing between each call individually with contract to the next or previous. BoundSums lets multiple occurrences of generate to excel like bundles with every other bundle blending individuals in a bundle at the similar amounts per bundle in generating occurrence.

Functions

Generate ([ParaPhrase]) String

Parameters

[ParaPhrase] String

An optional parameter to specify, a non-zero length, phrases ID that is used for as the Global ID for the specific LocalID generated. Asterisks are the default value if none is specified and the default or last call to Generate will be used.

Description

Generates a newly created LUID (local uniquely identifier). LUID's are of 36 character in length containing only hexadecimal characters from 0 to F and a total number of four dashes, dashes do not break hexadecimal formations of two characters on either end of the LUID and are random from beginning to end, they also do not come consecutive in sequence.
