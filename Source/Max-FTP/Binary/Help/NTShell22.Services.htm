NTShell22.Services ActiveX DLL



Identification

NTShell22.Services

Description

This object is an extension to manage requirements in properly starting and stopping NTService20.Controller built services and will operate for any normal hybrid service of NT based compatibility.

Functions

NetStart (ServiceName)

Parameters

ServiceName String

Specifies the unique name of the service as known to the system through its installation.

Description

Starts a Windows ME or Windows NT derivative based service through proper means of the system requirement and net.exe.

NetStop (ServiceName)

Parameters

ServiceName String

Specifies the unique name of the service as known to the system through its installation.

Description

Stops a Windows ME or Windows NT derivative based service through proper means of the system requirement and net.exe.

Optional event that is called when the object is terminated.
