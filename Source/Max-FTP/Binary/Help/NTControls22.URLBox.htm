NTControls22.URLBox ActiveX Control


Identification

NTControls22.URLBox

Description

An AutoType control that also contains a BrowseButton. Note: NTControls22 Objects are ActiveX Controls only for Visual Basic and are not available in the MaxIDE Environment.

Events

Change ()

Description

Occurs when the text has changed.

KeyDown (KeyCode, Shift)

Parameters

KeyCode Integer

The KeyCode of the key event that occurred.

Shift Integer

The shift state of the keyboard when the key event occurred.

Description

This event occurs when the user presses a key just before the KeyPress and KeyUp event.

Properties

AutoTypeCombo (Object)

Description

Gets the AutoType control of the URLBox.

hWnd (Long)

Description

Gets the hWnd of the control.

Functions

Reset ()

Description

Resets the control.
