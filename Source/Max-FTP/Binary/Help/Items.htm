Items

 

Item Menu

New Item

Adds a new item to the current project; showing a dialog to choose the item to add, by selecting it from a list of objects or files.

Open Item

Opens a new script editor to the code of the selected item in the project; allowing the editing of the code.

Examine Item

Opens a help document to the reference of the selected item in the project; information about specific objects properties methods and functions are the file that will be shown from the help of its item, respectively.

Delete Item

Deletes the selected item from the project; removing it permanently.

Rename Item

Invokes renaming of the selected item in the project; the name also refers to the object when addressing the object form code.
