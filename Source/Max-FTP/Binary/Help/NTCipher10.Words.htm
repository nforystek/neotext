NTCypher10.Words ActiveX DLL

Properties

Value (Word, Bit) Boolean

Parameters

Word Numerical

Specifies the byte, integer or long value that is the descriptive word to get or set the descriptor bit of.

Bit Bit

Specifies the bit of the word at which will be get or set, defined by the enumerator, accepts the numerical multiples of 2, such as 1,2,4,8,16,32,64,128,256,512…. Etc; up to the value 8388608.

Description

Gets or sets the Boolean bit value of a specified Bit on the numerical Word supplied.
