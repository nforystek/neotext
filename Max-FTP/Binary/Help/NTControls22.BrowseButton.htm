NTControls22.BrowseButton ActiveX Control


Identification

NTControls22.BrowseButton

Description

A small square folder button with or with out a border and/or mouse over border that can browse for File using the standard CommonDialog and also can be set to open browse windows for folder only directly initializing to default folders that include: (Desktop, ProgramFolder, Network, ControlPanel, Printers, MyDocuments, Favorites, RecyclingBin, MyComputer) Note: NTControls22 Objects are ActiveX Controls only for Visual Basic and are not available in the MaxIDE Environment.

Enumerators

BorderStyles

FixedSingle = 1
MouseOver = 2
None = 4

BrowseActions

None = -1
File = 0
Desktop = 1
ProgramFolder = 2
Network = 14
ControlPanel = 3
Printers = 4
MyDocuments = 5
Favorites = 6
RecyclingBin = 10
MyComputer = 13

Events

ButtonClick (BrowseReturn)

Parameters

BrowseReturn String

A string that contains the file or folder of the full path selected by the user after clicking the button, or contains a blank string to note the user clicked cancel.

Description

This even occurs after the user has clicked the browse button and either selected a path or canceled the selection.

Properties

BorderStyle (BorderStyles)

Description

Defines how the border of the browse button will show or react, possible values are bs_FixedSingle, bs_MouseOver, bs_None.

BrowseTitle (String)

Description

Sets the value to display as the title of the CommonDialog window when browsing for file.

BrowseAction (BrowseActions)

Description

Defines which browse action to use when the button is clicked. Possible values are: bws_None, bws_File, bws_Desktop, bws_ProgramFolder, bws_Network, bws_ControlPanel, bws_Printers, bws_MyDocuments, bws_Favorites, bws_RecyclingBin, bws_MyComputer.

FileFilter (String)

Description

A string value that defines which file filters will show in the CommonDialog window. [= description1 |filter1 |description2 |filter2...]

FileFilterIndex (Integer)

Description

Sets the index of the FileFilter to be selected by default.

FilterPath (String)

Description

Sets the default folder to display when the CommonDialog shows.

ButtonDimension (Integer)

Description

Defines the buttons height and width dimension.

ToolTipText (String)

Description

Set the ToolTipText of the button.

Enabled (Boolean)

Description

Sets whether the button is enabled or disabled.

Functions

Browse () String

Description

Invokes the browse action and returns the selected path, file, or blank string if the action was canceled by the user.
