Description

Above is a screenshot of Max-FTP's client window. A Max-FTP client window can connect to one or two sites at the same time. You can cut, copy, and paste files between any FTP connections; including between different windows. To load a new window, click on New Client Window under the Site menu. Each FTP connection has a ToolBar, Drive List, AddressBar, File/Log View, and StatusBar. Each window also has a menu for setup, navigation, and file actions. You can hide/un-hide certain elements of this window under the view menu.
