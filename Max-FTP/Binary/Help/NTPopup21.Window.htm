NTPopup21.Window ActiveX DLL



Identification

NTPopup21.Window

Description

This object is used to create popup windows.

Properties

ParentHWnd (Long)

Description

Specifies the hWnd value of the parent window to the popup window.

AllowInput (Boolean)

Description

Specifies whether the message portion of the window is input locked.

LinkText (String)

Description

Specifies the display text for the link portion on the popup window.

LinkURL (String)

Description

Specifies the URL for the link portion on the popup window.

Icon (Integer)

Description

Specifies the icon to be used on the popup window, their same values as the normal windows popup messages icon. 16, 32, 48, 64

Title (String)

Description

Specifies the title to be used on the popup window.

Message (String)

Description

Specifies the message to be used on the popup window.

AlwaysOnTop (Boolean)

Description

Specifies whether the popup window will remain above all other windows.

Visible (Boolean)

Description

Specifies whether the popup window is currently visible.

Functions

Show ()

Description

Shows the popup window changing the visible property.

Hide ()

Description

Hides the popup window changing the visible property.

Automation

Object:Init (self)

Parameters

self Object

A reference to the object.

Description

Optional event that is called when the object is initialized.

Object:Term (self)

Parameters

self Object

A reference to the object.

Description

Optional event that is called when the object is terminated.
