MaxIDE.Module Script API



Identification

MaxIDE.Module

Description

A modular script component that can be added to projects to organize code to custom desire. Multiple modules are allowed.

Automation

Module:First (Name)

Parameters

Name String

A strand reference that contains the name of the module.

Description

Optional event that is called before all object initializations.

Module:Default (Name)

Parameters

Name String

A strand reference that contains the name of the module.

Description

Optional event that is called before prior to project execution.

Module:Final (Name)

Parameters

Name String

A strand reference that contains the name of the module.

Description

Optional event that is called after all objects are terminated.
