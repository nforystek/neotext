NTShell22.Internet ActiveX DLL



Identification

NTShell22.Internet

Description

This object is used to post forms and retrieve data from web sites.

Functions

ResolveIP (Host) String

Parameters

Host String

The host name to resolve into its IP equivalent.

Description

Resolves an internet host name to its IP address.

LocalHost String

Description

Returns the host name of the local default network adapter.

PostToWebsite (HostServerAddress[, WebFilePath][, PostFormData][, AUthUsername][, AuthPassword][, CacheAndCookies][, SecureSocketLayer]) String

Parameters

HostServerAddress String

Specifies the server name. (Allows 'https://' to determine port 443 using SSL)

 

WebFilePath String

Specifies the web file on the server. Example: /default.asp

 

PostFormData String

Specifies the form data to send to the server Example: var1=val1&var2=val2

 

AuthUsername String

Specifies the authentication username to be used for low level verification communications such as NTLM policies.

 

AuthPassword String

Specifies the authentication password to be used for low level verification communications such as NTLM policies.

 

CacheAndCookies Boolean

Specifies whether to allow the use of cache and cookies during the post request. Sequential posts can be made from the same process allowing sessions to carry over and stay open.

 

SecureSocketLayer Boolean

Specifies whether to use secure sockets layer on port 443, this overwrites the use of 'http://' or 'https://' in the HostServerAddress.

Description

This function sends out a web request and returns the data the server returned.

OpenWebsite (Weblink) Boolean

Parameters

Weblink String

Specifies the server name.

Description

Attempts to open a website using default browser.

URLEncode (encodeString) String

Parameters

encodeString String

Specifies the data to encode with escape codes.

Description

Encodes a string with proper URL escape codes for malformed addresses and passing web data through parameters.

URLDecode (encodedString) String

Parameters

encodedString String

Specifies the data with escape codes to un-encode.

Description

Decodes a string of proper URL escape codes to reform proper addresses and web parameter data.
